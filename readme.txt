== Description ==

Elementor addon to create an custom mouse

== Installation ==

1. Upload 'qanva-custom-mouse-for-elementor.zip' through Wordpress 'Plugins' installation menu
2. Activate the plugin through the 'Plugins' menu in WordPress

== Screenshots ==

1. Widget for the custom mouse. 
2. Settings like color, vover-color, disc-color, size and disc-size and preview.
3. Live on a website (example properties)
